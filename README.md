# xamarin-android-docker

[![dockeri.co](http://dockeri.co/image/sabvente/xamarin-android-docker)](https://hub.docker.com/r/sabvente/xamarin-android-docker/)

Docker image for building Xamarin Android apps

### Compatibility
 - Android SDK 27
 - .NET Core SDK 2.2
 - msbuild 16.0.0.0
 - mono 5.18.0.225
